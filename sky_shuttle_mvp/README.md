# 天际飞梭：卡牌冲刺 (Sky Shuttle: Card Dash)

一款竖屏无尽跑酷卡牌游戏，使用 Godot 4.6 开发。

## 游戏特色

- 🎮 竖屏操作，适合手机游玩
- 🃏 卡牌系统（跳跃/滑铲/投掷）
- ⚡ 子弹时间机制
- 📈 多阶段难度递增
- 🎯 策略性障碍物应对

## 操作方式

| 操作 | 说明 |
|------|------|
| ← 按钮 | 向左移动 |
| → 按钮 | 向右移动 |
| 点击卡牌 | 使用对应技能 |

## 卡牌系统

- **跳跃卡牌** (绿色): 对跳跃障碍无敌 0.7秒，消耗 2 能量
- **滑铲卡牌** (蓝色): 对滑铲障碍无敌 1.0秒，消耗 2 能量
- **投掷卡牌** (橙色): 发射子弹破坏木箱，消耗 1 能量

## 技术栈

- Godot Engine 4.6
- GDScript
- 支持 Web/HTML5 导出
